## Setup:

1.  Clone the repository.
2.  Install dependencies.

        yarn install

3.  Setup environment variables in `.env` file.

        PORT=
        MONGO_CONNECTION_URL=
        APP_BASE_URL=
        SMTP_HOST=
        SMTP_PORT=
        MAIL_USER=
        MAIL_PASS=
        ALLOWED_HOSTS=

4.  Run the server.

        yarn start
